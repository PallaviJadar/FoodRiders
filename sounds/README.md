# Audio Files for Siren System

## Required Files

Place the following audio files in this directory:

1. **admin-siren.mp3** - Loud alarm for admin alerts
2. **delivery-alert.mp3** - Alert tone for delivery partners
3. **notification.mp3** - Soft chime for user notifications
4. **success.mp3** - Success confirmation sound

## Temporary Solution

For testing, you can use any MP3 files with these names. The system will work with any audio format supported by browsers (MP3, WAV, OGG).

## Free Sound Resources

- **Freesound.org** - https://freesound.org/
- **Zapsplat** - https://www.zapsplat.com/
- **Mixkit** - https://mixkit.co/free-sound-effects/

## Quick Test

You can use simple beep sounds or even silent files for initial testing. The important part is that the files exist with the correct names.
